AI Coding Starter Kit helps beginners prepare a Mac for AI coding tools through guided pages, copied commands, read-only checks, troubleshooting branches, and a local setup assistant.

Terminal asks for Password. Is that normal?

It can be normal if you intentionally started an official installer that clearly says it needs administrator permission. macOS hides password typing in Terminal, so you may not see characters or dots while typing.

A command asks for y. Should I continue?

Read the lines above the prompt first. Continue only if the installer is official and the listed changes match the step you chose.

Does the app install tools for me?

No. The app can explain how to install optional tools and can copy commands, but the user remains in control of every installation.

What should I include in a support request?

Include macOS version, app version and build, which page you were using, non-secret Terminal output, and whether you are using Apple Silicon or Intel Mac. Never send passwords, API keys, tokens, cookies, recovery codes, private SSH keys, or payment details.
