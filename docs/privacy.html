Privacy Policy

Effective date: 2026-06-09

AI Coding Starter Kit is designed as a local beginner setup guide for macOS.

Data Collection

The app does not collect, sell, or track personal data. It stores small app-local preferences on the device, such as selected language and local feature state.

Network Access

The app can perform network readiness checks against public developer service endpoints. StoreKit uses Apple's purchase infrastructure for in-app purchases and restore purchase actions.

Clipboard, Terminal, And Files

The app can copy user-selected setup commands to the macOS clipboard, open Apple's Terminal app when the user presses Open Terminal, and export a Markdown checklist after the user chooses a save location.

Local Assistant

The local assistant is intended for setup guidance. Users should not paste passwords, API keys, tokens, cookies, recovery codes, private keys, payment details, or private account information into the assistant.

Contact

For support, use the support URL listed on the App Store product page.
